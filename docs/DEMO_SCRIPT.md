# Demo Script - Smart HR Onboarding Assistant

## Overview
Duration: 5-7 minutes
Goal: Showcase the power of agentic AI in HR onboarding

## Demo Scenario
**New Hire**: Sarah Chen, Software Engineer
**Start Date**: Monday, November 18, 2024

---

## Act 1: HR Creates Onboarding (1 min)

**Narrator**: "Meet Sarah Chen, our new software engineer starting Monday..."

**Demo Steps**:
1. Login to admin portal as HR
2. Click "Create New Onboarding"
3. Fill in Sarah's details
4. Click "Start Onboarding"
5. Show automated workflow triggered

**Key Points**:
- One-click creation
- Automatic workflow orchestration
- Real-time status updates

## Act 2: Behind the Scenes - watsonx Orchestrate Magic (2 min)

**Show workflow visualization**:
